## 📌 About

This project is a simple interactive CLI toolkit built with Bash scripting.

It provides a menu-driven interface that allows users to execute common Linux system commands such as:

- Display network interfaces
- Show routing table
- Check current user
- View disk usage

The script includes input validation, loop control, and user confirmation prompts to improve usability and reliability.

## 🎯 Purpose

This project was created as a practical exercise to improve Bash scripting skills, focusing on:

- Conditional statements (`if`, `case`)
- Loops (`while`)
- Input validation
- Command execution
- Basic CLI user interaction

## 🛠 Technologies

- Bash
- Linux CLI utilities
- iproute2
- Core GNU utilities
